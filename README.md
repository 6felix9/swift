# Financial Advisor AI Training

This is a training tool for financial advisors to evaluate their referral-seeking skills based on the "Referral Readiness Evaluation Criteria" provided to them.

## How it works

1. The financial advisor is asked to role-play as a client and interact with the AI voice assistant.
2. The AI voice assistant will use the "Referral Readiness Evaluation Criteria" to evaluate the financial advisor's referral-seeking skills.
3. The financial advisor is asked to role-play as a client and interact with the AI voice assistant.
